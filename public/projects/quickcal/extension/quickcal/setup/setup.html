How to Generate a Gemini API Key

Follow these steps to create an API key for Google's Gemini API and use it with this Chrome extension:

Step 1: Go to Google AI Studio

Visit the Google AI Studio website. Log in with your Google account if prompted. The page should look like the screenshot below.

Step 2: Click "Get API Key" & Agree to Terms

Click on "Get API Key". Then, agree to the Terms and Conditions and click "Continue".

Step 3: Create API Key

Now that you've enabled the Gemini API, click on "Create API Key". Google will take a few seconds to generate the key.

Step 4: Copy API Key

A pop-up should appear letting you copy your API Key. Copy the key and paste it into the QuickCal extension.


Step 5: Use QuickCal!

Once your API key is validated, use the extension! Highlight any event description to quickly add it to your calendar.
