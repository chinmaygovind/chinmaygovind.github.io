ROBOT TOUR
Science Olympiad Autonomous Robotics Project
This project documents my journey building, coding, and competing with a custom autonomous robot for Science Olympiad's Robot Tour event. Built for durability, programmed for precision, and tested for real-world chaos.
BUILD

Meet the Robot

  • Chassis: Custom-built for durability and modularity.
  • Brains: Arduino-based, programmed in C++.
  • Sensors: Encoders, gyros, and ultrasonic sensors for navigation and obstacle detection.
  • Power: LiPo battery for reliable performance.
